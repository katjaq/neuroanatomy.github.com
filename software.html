Web applications
  • BrainSpell Neuroimaging annotation http://brainspell.org
  • BrainCatalogue Collaborative comparative neuroanatomy http://braincatalogue.org
  • Microdraw Interactive, collaborative, online, multi-scale viewer and annotation tool http://microdraw.pasteur.fr
Desktop applications
  • CoactivationMap Meta-analytic atlas of human functional coactivation http://neuroanatomy.github.io/CoactivationMap
  • NeuroanatomyRAMONES A series of tools for computational neuroanatomy. For organising data lists CollectionRAMON, for viewing and segmenting volume data StereotaxicRAMON, for viewing surface data SurfaceRAMON, and some others to be added in the future
  • QCApp Quality control tools for large neuroimaging cohorts, for lobar volumes QCApp-vlob and subcortical structures QCApp-vsub
  • SnipPeep Fast viewer for copy-number variations in genome-wide genotyping data
    • http://neuroanatomy.github.io/SnipPeep
  • MeshGeometry Command line toolbox for processing meshes
  • MeshSurgery Simple editor for mesh data with interactive geometry smoothing
  • EditMesh EditMesh is ugly, but it lets you do tricks that you cannot do with other tools
